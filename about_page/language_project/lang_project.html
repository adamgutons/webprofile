Java: Language Structures Project

Description of Music Key Generator

For my mini-project I coded a program that will generate musical keys. User will input a key signature (C minor, A major, flats/sharps included) and the program will return the notes in the key, the triad that will make the chord (ex the first, third, and fifth notes of the key). The program will also provide the relative major or minor key.

This project was an exploration of the HashMap java class. Most of the knowledge that I acquired was thanks to the youtube video below. A HashMap is a very useful tool for storing and calling on associated variables < V, K, >). Data is stored in a linked list of nodes also known as "buckets". The put(); function stores variable data in the nodes, and the get(); function calls on data.

----------------------------------

The Heart of the Matter

"Hashing" in the terms of the method refers to converting a string (text) character to a value that represents the string. In essence, the string is turned to bytes. This makes the data easier to locate. It is helpful that every java object is associated with a hashcode(); method that returns an objects hashcode value.

The put(); method will determine a node location, or index, to place the byte data by dividing the objects hash value and the max value of the hash range. The formula looks like this: index = hash&(n-1) where n is the maximum node value (again, in a standard hashmap n=16).

The get(); method will attempt to map the hashcodes of the V and K variables and once a hashcode match is found, get(); will match the K(key) values and if it determines another successful match it will return the stored string data.
